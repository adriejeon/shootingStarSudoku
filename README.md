# Shooting Star Sudoku

별이 떨어지는 수학 퍼즐 게임

## 프로젝트 소개

Shooting Star Sudoku는 전통적인 스도쿠 게임에 별의 움직임과 시각적 효과를 추가한 혁신적인 퍼즐 게임입니다.

## 기능

- 클래식 스도쿠 규칙
- 별이 떨어지는 시각적 효과
- 직관적인 사용자 인터페이스
- 다양한 난이도 설정

## 개발 환경

- React
- TypeScript
- CSS3

## 설치 및 실행

```bash
npm install
npm start
```
